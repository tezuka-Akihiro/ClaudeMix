## 🎯 Usage Guidelines（使用ガイドライン）

### Color Usage（色の使い分け）
| 色 | 主な用途 | 使ってはいけない場所 |
|:---|:---|:---|
| Cyan | ボーダー、構造、プライマリアイコン | ステータス表示 |
| Green | 完了、成功、ポジティブフィードバック | エラーメッセージ |
| Red | 未完了、エラー、警告 | 成功メッセージ |
| Blue | 選択中、ホバー、フォーカス | ステータス表示 |
| Yellow | 進行中、注目 | エラー表示 |
| Gray | 背景、非アクティブ、区切り | アクションボタン |

### Spacing Usage（余白の使い分け）
| サイズ | 主な用途 |
|:---|:---|
| xs (4px) | アイコンとテキストの間 |
| sm (8px) | インラインメントの間 |
| md (12px) | コンパクトなパディング |
| base (16px) | 標準的なパディング |
| lg (24px) | セクション内の区切り |
| xl (32px) | セクション間の区切り |
| 2xl (48px) | ページレベルの区切り |
