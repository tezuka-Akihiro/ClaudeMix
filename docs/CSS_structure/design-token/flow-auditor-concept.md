サービス名: flow-auditor
テーマ: cyberpunk-terminal

## 🎯 世界観のキーワード

### 1. 監査人（Auditor）
- **意味**: システムの状態を客観的に検査・報告する存在
- **デザインへの反映**:
  - クリーンで無駄のないレイアウト
  - 状態を一目で把握できる明確な色分け（完了/未完了）
  - データ主導の視覚化（進捗率、チェックリスト）

### 2. チェックポイント（Checkpoint）
- **意味**: 通過すべき検証ポイント、セーブポイント
- **デザインへの反映**:
  - ノード形式の表示（ゲームのマップのように）
  - 完了/未完了の明確な視覚的フィードバック
  - フローチャート的な接続線

### 3. ターミナル/コマンドライン（Terminal）
- **意味**: 開発者に馴染みのあるCLIインターフェース
- **デザインへの反映**:
  - モノスペースフォント
  - シャープなエッジ（角丸なし）
  - ターミナルカラーパレット（cyan, green, red, yellow）

### 4. サイバーパンク（Cyberpunk）
- **意味**: 近未来的、ネオン、ダークモード、グロー効果
- **デザインへの反映**:
  - ダーク背景（黒〜濃いグレー）
  - ネオンカラーのアクセント（cyan, magenta, green）
  - グロー/シャドウ効果（box-shadow）
  - グリッドライン（デジタル感）

## 🖼️ ムードボード

### 参考イメージ
- **flow-auditorの継承**: 既存のcyberpunk-terminalテーマを基盤とする
- **ターミナルUI**:
  - Vim、Emacsのようなキーボード駆動
  - htop、btopのようなシステム監視ツールのUI
- **ゲームUI**:
  - サイバーパンク2077のメニューUI
  - デウスエクス シリーズのハッキングUI

### 色の雰囲気
- **ベース**: 深い黒（#000000）〜濃いグレー（#1a1a1a）
- **構造色**: Cyan（#22d3ee）- 枠線、ボーダー、構造要素
- **アクション色**: Green（#4ade80）- 完了、成功、ポジティブ
- **警告色**: Red/Pink（#f87171, #ec4899）- 未完了、エラー、警告
- **インタラクション色**: Blue（#60a5fa）- 選択中、ホバー、フォーカス

## 📐 デザイントーン分析

### 1. 構造（Structure）
- **レイアウト**: グリッドベース、整然とした配置
- **間隔**: 適度な余白（8px基準のスペーシングシステム）
- **コンテナ**: 明確なボーダー（2px solid）、角丸なし
- **階層**: セクションごとに視覚的に分離（ボーダー、背景色の微妙な差）

### 2. 余白（Spacing）
- **密度**: 中〜高密度（情報量重視、ダッシュボード的）
- **パディング**: 内側の余白は控えめ（0.75rem〜1rem）
- **マージン**: 要素間は明確に分離（1rem〜2rem）

### 3. タイポグラフィ（Typography）
- **フォントファミリー**: モノスペース（Courier New, Monaco, Consolas）
- **階層**: 明確なサイズ差（見出し: 1.25rem, 本文: 1rem, 補足: 0.875rem）
- **ウェイト**: Medium〜Bold（視認性重視）
- **文字色**: 高コントラスト（WCAG AA準拠）

### 4. カラートーン（Color Tone）
- **彩度**: 高彩度（鮮やかなネオンカラー）
- **明度**: ダーク背景に映えるミッドトーン〜ハイライト
- **コントラスト**: 強いコントラスト（黒背景 + 明るいアクセント）
- **グロー効果**: box-shadowで発光感を演出（0 0 10px color）

### 5. インタラクション（Interaction）
- **ホバー**: グロー強化、ボーダーカラー変化
- **選択**: ボーダーカラー変化（cyan → blue）、グロー効果追加
- **トランジション**: スムーズ（0.3s cubic-bezier）
- **フィードバック**: 即座に視覚的変化

## 🎨 デザインの方向性まとめ

**flow-auditorは、「開発フローの監査人」として、以下の価値を視覚的に表現します:**

1. **信頼性**: クリーンで整然としたレイアウト、データ主導の表示
2. **即時性**: 一目で状態が把握できる明確な色分け
3. **専門性**: 開発者に馴染みのあるターミナル風デザイン
4. **未来感**: サイバーパンクの世界観を継承したネオンカラーとグロー効果

**flow-auditorとの関係**:
- 同じcyberpunk-terminalテーマを共有
- カラーパレットとグロー効果を継承
- ただし、flow-auditorは「監査」という役割に特化し、よりクリーンで整然としたレイアウトを採用
