## blog-article-writer.md.old

### 削除日
2025-12-19

### 削除理由
ブログ執筆プロンプトをフェーズ別に分割するため、以下の3つのファイルに分割しました：

1. **blog-metadata-writer.md** (プロンプト): タグ・カテゴリ・Frontmatter選定
2. **blog-introduction-writer.md** (スキル): 導入基本構成の執筆（恐怖型BAB法の5パート）
3. **blog-body-writer.md** (プロンプト): 本文執筆（テンプレート別の構成指示を含む）

### 問題点
- タグ・カテゴリ選定のみに特化しており、テンプレートや導入基本構成の指示がなかった
- 実際のブログ執筆フローに対応していなかった
- 導入基本構成の「精密な人間行動促進テクニック」を扱えていなかった

### 影響範囲
- `.claude/prompts/blog-article-writer.md` を削除
- 新しい3つのファイルに置き換え

### 代替手段
今後は以下のフローでブログを執筆します：

1. `blog-metadata-writer` プロンプトでメタデータを作成
2. `blog-introduction-writer` スキルで導入部を執筆
3. `blog-body-writer` プロンプトで本文を執筆

### 再発防止策
- プロンプトは単一責任の原則に従い、1つのタスクに特化させる
- 複雑なフローは複数のプロンプト/スキルに分割する
- 精密なテクニックが必要な部分（導入基本構成）はスキルとして独立管理
