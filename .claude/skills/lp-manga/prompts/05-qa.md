# Phase 5: 品質検証プロンプト

## AI役割定義

あなたは品質管理者です。
生成されたLP漫画の画像素材に対して、品質チェックを実施してください。

## 前提条件

Phase 4で生成した以下が存在すること：

- 全コマの画像素材（OK判定済み）
- 生成ログ・アセット管理表

## チェックリスト

### 1. 構造・整合性チェック

#### キャラクターの一貫性

| チェック項目 | 確認方法 | 判定 |
| :--- | :--- | :--- |
| 髪型が全コマで統一されている | 全画像を並べて比較 | OK / NG |
| 髪色が全コマで統一されている | 同上 | OK / NG |
| 服装が全コマで統一されている | 同上 | OK / NG |
| 眼鏡等のアクセサリーが維持されている | 同上 | OK / NG |

**NG時の対処**: Phase 4に戻り、該当コマを三面図を参照して再生成

#### 指・四肢の整合性

| チェック項目 | 確認方法 | 判定 |
| :--- | :--- | :--- |
| 指の本数が正常（5本） | 手が見えるコマを確認 | OK / NG / N/A |
| 関節の曲がり方が自然 | 全身が見えるコマを確認 | OK / NG / N/A |
| 腕・足の接続が自然 | 同上 | OK / NG / N/A |

**NG時の対処**:

- 軽微 → 画像編集ツールで修正
- 重大 → 「手を描かない構図」で再生成

#### 視線誘導

| チェック項目 | 確認方法 | 判定 |
| :--- | :--- | :--- |
| コマの配置が読みやすい | 全体を通して読んでみる | OK / NG |
| 吹き出し用の余白が確保されている | 各コマの余白を確認 | OK / NG |
| 視線の流れが自然（Z型 or F型） | 同上 | OK / NG |

**NG時の対処**: 構図を変更して再生成、またはDTP段階で調整

---

### 2. コンプライアンス・媒体チェック

#### 文字の可読性

| チェック項目 | 確認方法 | 判定 |
| :--- | :--- | :--- |
| 吹き出し用スペースが十分 | 実際にテキストを配置してみる | OK / NG |
| スマホ表示で14px以上相当 | デバイスプレビューで確認 | OK / NG |

**NG時の対処**: 余白を広げて再生成、またはフォントサイズ調整

#### 商標・権利

| チェック項目 | 確認方法 | 判定 |
| :--- | :--- | :--- |
| 意図しないロゴが映り込んでいない | 背景を注視 | OK / NG |
| 既存キャラクターに似すぎていない | 全体の印象を確認 | OK / NG |
| 実在の人物に似すぎていない | 同上 | OK / NG |

**NG時の対処**: 該当箇所を修正して再生成
