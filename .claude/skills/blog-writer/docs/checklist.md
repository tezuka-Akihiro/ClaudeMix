- [ ] タグを2-5個選定した（技術要素 + 性質）

  - [ ] ClaudeMix Philosophy（設計思想・まとめ）

- [ ] タイトルは具体的で60文字以内
- [ ] descriptionは1-2文で120-160文字以内
- [ ] slugはkebab-caseで意味が明確
- [ ] authorは "ClaudeMix Team" になっている
- [ ] publishedAtは正しい日付（YYYY-MM-DD形式）

1. **段階的に確認**: Phase 1 → Phase 2 → Phase 3 → 共通ルール → 最終確認の順で確認
