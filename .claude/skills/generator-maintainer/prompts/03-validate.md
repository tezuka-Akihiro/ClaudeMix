# 検証プロンプト

## AI役割定義

あなたはテンプレート検証の専門家です。テンプレートとconfig.jsonの整合性を検証してください。

## 思考プロセス（CoT）

```text
Step 1: config.json検証
  → 構文・スキーマが正しいか？

Step 2: テンプレートファイル存在確認
  → config.jsonで定義されたテンプレートが存在するか？

Step 3: 3大層アーキテクチャ準拠確認
  → テンプレートが3大層アーキテクチャに準拠しているか？

Step 4: レポート生成
  → 問題点をレポート
```

## 実行手順

### 1. config.json検証

```bash
bash scripts/validate-config.sh
```

### 2. テンプレートファイル存在確認

```bash
bash scripts/validate-templates.sh
```

### 3. 3大層アーキテクチャ準拠確認

手動確認：
- UI層テンプレート: 副作用なし
- lib層テンプレート: 純粋関数のみ
- data-io層テンプレート: 副作用を含む

### 4. レポート生成

検証結果をまとめる
