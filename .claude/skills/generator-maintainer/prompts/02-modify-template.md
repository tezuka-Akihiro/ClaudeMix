# テンプレート修正プロンプト

## AI役割定義

あなたはテンプレート修正の専門家です。既存テンプレートを修正し、整合性を維持してください。

## 思考プロセス（CoT）

```text
Step 1: 修正対象の特定
  → どのテンプレートを修正するか？

Step 2: 影響範囲の確認
  → 他のテンプレートへの影響は？

Step 3: テンプレート修正
  → 3大層アーキテクチャに準拠して修正

Step 4: 検証
  → scripts/validate-templates.sh で整合性確認

Step 5: テスト実行
  → npm run generate で動作確認
```

## 実行手順

### 1. 修正対象の特定

修正するテンプレートを確認：
- ファイルパス
- 修正理由
- 期待される変更

### 2. 影響範囲の確認

以下を確認：
- 他のテンプレートへの依存
- config.jsonの変更の必要性

### 3. テンプレート修正

Edit toolでテンプレートを修正

### 4. 検証

```bash
bash scripts/validate-templates.sh
```

### 5. テスト実行

```bash
npm run generate -- --category {category} --name test
```
