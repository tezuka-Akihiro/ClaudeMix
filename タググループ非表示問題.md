# 不具合調査・解消テンプレート

# 1. 不具合の概要
## 発生事象
*   ブログ記事一覧ページのフィルターパネルで、タグがグループ化されて表示されない。タググループ機能全体がUIに表示されていないように見える。

## 再現手順
1.  `npm run dev` で開発サーバーを起動する。
2.  ブラウザでブログ記事一覧ページ (`/blog`) にアクセスする。
3.  「フィルター」ボタンをクリックしてフィルターパネルを開く。

## 期待される動作
*   フィルターパネル内に、「Remix」「Cloudflare」などのグループ見出しが表示され、その下に関連するタグボタンがグリッド表示される。

## 実際の動作
*   タググループの見出しやグループ化されたレイアウトが表示されず、機能がまるごと抜け落ちているように見える。
*   CLIには `Error: No route matches URL "/.well-known/appspecific/com.chrome.devtools.json"` というエラーが表示されているが、これは直接の原因ではない可能性が高い。

## 環境
*   **OS**: Windows 11
*   **ブラウザ**: Chromium
*   **Node.js**: (package.jsonに記載なし)
*   **Remix**: v2.17.1
*   **Vite**: v6.3.6
