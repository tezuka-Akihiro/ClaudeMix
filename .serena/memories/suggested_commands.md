# Suggested Commands

## Development Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production  
- `npm run typecheck` - Run TypeScript type checking
- `npm run lint` - Run linting
- `npm start` - Start production server

## System Commands (Windows)
- `dir` - List directory contents (equivalent to `ls`)
- `cd` - Change directory
- `findstr` - Search text in files (equivalent to `grep`)
- `where` - Find file locations (equivalent to `which`)
- `type` - Display file contents (equivalent to `cat`)

## Git Commands
- `git status` - Check repository status
- `git add .` - Stage all changes
- `git commit -m "message"` - Commit changes
- `git push` - Push to remote

## Package Management
- `npm install` - Install dependencies
- `npm install <package>` - Install specific package
- `npm run <script>` - Run package.json script
